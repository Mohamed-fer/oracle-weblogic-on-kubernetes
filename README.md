\# Oracle WebLogic on Kubernetes



\## Overview



Production-like Kubernetes platform built using:



* Kubernetes
* Oracle WebLogic
* NFS Storage
* Persistent Volumes
* Linux



\## Infrastructure



* 1 Kubernetes Master Node
* 3 Kubernetes Worker Nodes
* NFS Storage Server



\## Features



* High AvailabilityP
* Persistent Storage
* WebLogic Domain Deployment
* Kubernetes Native Management
