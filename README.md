\# Oracle WebLogic on Kubernetes



\## Overview



This project demonstrates the deployment of Oracle WebLogic Server on a Kubernetes cluster using kubeadm.



It simulates a production-like environment with a multi-node Kubernetes cluster, NFS storage, and persistent volume management.



\## Infrastructure



\- 1 Kubernetes Master Node

\- 3 Kubernetes Worker Nodes

\- NFS Storage Server



\## Technologies



\- Kubernetes (kubeadm)

\- Oracle WebLogic Server

\- Docker

\- NFS (Network File System)

\- Linux



\## Features



\- High Availability deployment

\- Persistent storage using NFS

\- WebLogic domain deployment on Kubernetes

\- Kubernetes-native workload management

\- Persistent Volumes (PV) and Persistent Volume Claims (PVC)
